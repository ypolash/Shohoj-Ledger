# Shohoj Ledger

Shohoj Ledger is a modern SaaS Multi-Tenant ERP/CRM platform.

Mission

Build a scalable ERP platform supporting unlimited companies while preserving backward compatibility.

Core Modules

- Finance
- CRM
- HR
- Payroll
- Attendance
- Projects
- Inventory
- Reports

Core Principles

- Never break existing functionality.
- Extend instead of rewrite.
- Backward compatibility first.
- Production stability over new features.

Current Goal

Safely migrate from Single Tenant to Multi Tenant.
